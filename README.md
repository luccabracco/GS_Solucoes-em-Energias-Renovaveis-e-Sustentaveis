##  Introdução
Este projeto é uma solução computacional desenvolvida para a Global Solution de Soluções em Energias Renováveis e Sustentáveis. O objetivo do sistema é monitorar as condições operacionais de uma missão espacial, focando no gerenciamento de energias renováveis e sustentáveis para garantir a segurança e eficiência do módulo.

##  Funcionalidades Implementadas
O script Python simula em tempo real as variações em um ambiente espacial e executa as seguintes tarefas:
* **Monitoramento de Dados:** Leitura constante de temperatura, nível de energia (baterias/painéis) e status de comunicação.
* **Geração de Alertas:** Identificação autônoma de anomalias (ex: superaquecimento, perda de sinal).
* **Tomada de Decisão:** Respostas automatizadas para situações críticas (ex: desligar sistemas não essenciais se a energia cair abaixo de 30%).
* **Visualização:** Interface no terminal para acompanhamento ciclo a ciclo da missão.
